ShivChat v1.1

ShivChat is a free IRC Chat system for Unity3D projects, using the 3rd party open source thresher C# IRC library (http://thresher.sourceforge.net/).
ShivChat offer basic IRC funcionality, that includes multiple channel chat, pm messaging system and a user block system.

By default, the system offers up to 4 tabs (channels), its possible to add or remove them.
